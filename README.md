Roso 是一个面向鲜花零售和花店运营的综合服务平台，由微信小程序用户端、Vue 商家后台和 Spring Boot 后端组成。用户端支持鲜花浏览、AI 推荐、购物车、微信支付、预约排队、活动报名、积分签到、优惠券兑换和消息通知；商家端支持商品管理、订单管理、活动发布、积分券配置、预约处理、消息中心、库存预警、数据看板和 AI 运营助手；后端使用 Spring Boot 3、MyBatis、MySQL、Redis、RabbitMQ、WebSocket、七牛云、微信支付和 AI 服务构建完整业务闭环，并通过 Redis Lua、订单状态机、支付幂等、限流熔断和消息队列保障高并发场景下的系统稳定性。
