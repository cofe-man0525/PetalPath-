Roso 是一个面向鲜花零售和花店运营的综合服务平台，由微信小程序用户端、Vue 商家后台和 Spring Boot 后端组成。用户端支持鲜花浏览、AI 推荐、购物车、微信支付、预约排队、活动报名、积分签到、优惠券兑换和消息通知；商家端支持商品管理、订单管理、活动发布、积分券配置、预约处理、消息中心、库存预警、数据看板和 AI 运营助手；后端使用 Spring Boot 3、MyBatis、MySQL、Redis、RabbitMQ、WebSocket、七牛云、微信支付和 AI 服务构建完整业务闭环，并通过 Redis Lua、订单状态机、支付幂等、限流熔断和消息队列保障高并发场景下的系统稳定性。

## 4. 后端业务模块

### 4.1 登录鉴权模块
Sa-Token + Redis 实现小程序/管理员登录鉴权，拦截器统一校验 Token。

### 4.2 用户与地址模块
用户信息管理、积分/订单统计，地址增删改查及默认地址管理。

### 4.3 商品模块
商品增删改查、分页筛选、上下架、库存预警、图片上传（七牛云）。

### 4.4 购物车模块
加入购物车、数量修改、结算前价格/库存校验。

### 4.5 订单模块
Snowflake 生成订单号，订单主表+明细表，状态机流转，RabbitMQ 延迟队列实现 30 分钟超时关闭。

### 4.6 支付模块
微信支付 V3 统一下单+回调验签，支付流水表幂等处理，Redis 分布式锁防重复回调。

### 4.7 预约排队模块
Redis 容量控制 + Redisson 阻塞队列 + RabbitMQ 异步处理 + WebSocket 实时推送结果。

### 4.8 活动模块
活动发布/报名，防重复报名，Redis 缓存热门活动，定时任务推送活动提醒。

### 4.9 积分签到模块
每日签到、连续签到奖励，Redis 记录签到状态，MySQL 事务更新积分。

### 4.10 优惠券与积分兑换模块
Redis Lua 原子扣减库存，唯一索引防重复兑换，定时任务处理过期券。

### 4.11 秒杀抢购模块
Redis 预热库存，Lua 原子校验 + 一人一单，RabbitMQ 异步下单，乐观锁兜底。

### 4.12 消息通知模块
RabbitMQ 发布事件，WebSocket 实时推送，Redis 缓存未读数。

### 4.13 AI 推荐与运营助手模块
Spring AI + 外部 Python 服务，实现鲜花推荐、祝福语生成、商品文案辅助，Langchain,Aiagent智能体，RAG,MCP.

### 4.14 图片资源模块
七牛云 SDK 上传/替换/删除图片，UUID 命名，定时清理临时图片。

### 4.15 数据看板模块
MySQL 聚合查询销售额/订单/热销商品等，Redis 缓存统计结果。

### 4.16 系统配置模块
MySQL 配置表 + Redis 缓存，支持店铺/预约/积分等规则动态配置。

### 4.17 工程规范模块
全局异常处理、统一返回结果、Jakarta Validation 参数校验、Knife4j 接口文档、Docker Compose 一键启动。

项目描述：
基于 Spring Boot 3 + Java 17 开发的鲜花服务平台后端系统，统一支撑微信小程序用户端和 Vue 商家后台。
系统支持微信登录、商品管理、购物车、订单支付、地址管理、积分签到、优惠券兑换、活动报名、预约排队、秒杀抢购、AI 鲜花推荐、消息通知和商家数据看板等功能。后端采用 MyBatis + MySQL 持久化业务数据，Sa-Token + Redis 实现登录鉴权，RabbitMQ 处理预约、支付、通知等异步任务，WebSocket 实现实时消息推送，七牛云管理图片资源，并接入微信支付 V3 和 AI 服务。

技术亮点：
1. 基于 Sa-Token + Redis 实现用户端和商家端统一鉴权，通过拦截器统一校验接口访问权限。
2. 封装 Redis 缓存工具类，支持缓存回填、空值缓存、互斥锁重建缓存和列表缓存，降低热点数据访问数据库的压力。
3. 设计 Redis/Redisson + RabbitMQ 的预约排队机制，对预约请求进行容量限制、重复校验和异步落库，并通过 WebSocket 实时推送预约结果。
4. 支付模块接入微信支付 V3，使用订单状态机、支付流水表、回调验签和幂等处理保证支付流程可靠性。
5. 使用 RabbitMQ 延迟队列实现订单超时自动关闭，避免未支付订单长期占用库存。
6. 积分兑换和秒杀模块使用 Redis Lua 实现库存原子扣减，并结合唯一索引和数据库事务防止超发和重复兑换。
7. 基于 RabbitMQ + WebSocket 构建消息中心，在支付成功、预约成功、活动报名等场景实时通知用户端和商家端。
8. 接入七牛云对象存储，实现商品、活动、积分券图片上传、替换和删除。
9. 集成 AI 推荐与运营助手，根据用户预算、节日和送礼对象生成鲜花推荐和祝福语，并支持商家端运营文案生成。
